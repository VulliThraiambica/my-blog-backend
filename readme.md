# ⚙️ Blog Backend

A backend server for the Blog Application built using Node.js, Express.js, and MongoDB Atlas.

The backend manages authentication, article creation, article management, user roles, and database operations through REST APIs.


##  Features

- User Authentication using JWT
- Role-Based Authorization
- Create Blog Articles
- Read Articles
- Edit Articles
- Delete/Deactivate Articles
- Comment System
- Protected APIs
- MongoDB Atlas Integration
- Error Handling
- CORS Configuration

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs
- dotenv
- cookie-parser
- CORS
